# Tic Tac Toe Game

## Description

This is a simple implementation of the classic Tic Tac Toe game. It is built using HTML, CSS, and JavaScript, with a minimalist design.

## Features

- Responsive design
- Minimalist UI
- Green X's, white squares, and red buttons
- Reset button to start a new game

## Installation

No installation required. Just open the `index.html` file in a web browser.

## Usage

1. Open `index.html` in your preferred web browser.
2. Click on the squares to make your move.
3. The game will automatically indicate a winner or a draw.
4. Click the "Reset" button to start a new game.
